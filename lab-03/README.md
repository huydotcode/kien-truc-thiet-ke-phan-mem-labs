Lab-03 - Design Patterns

## Chạy các bài tập thực hành Design Pattern

### 1. Composite Pattern

- **Chạy bài Hệ thống File (File System):**
  ```bash
  npx ts-node 1-composite/file-system.ts
  ```
- **Chạy bài Giao diện người dùng (UI Components):**
  ```bash
  npx ts-node 1-composite/ui-components.ts
  ```

### 2. Observer Pattern

- **Chạy bài Thị trường Cổ phiếu (Stock Market):**
  ```bash
  npx ts-node 2-observer/stock-market.ts
  ```
- **Chạy bài Quản lý Công việc (Task Management):**
  ```bash
  npx ts-node 2-observer/task-management.ts
  ```

### 3. Adapter Pattern

- **Chạy bài Chuyển đổi dữ liệu (XML sang JSON):**
  ```bash
  npx ts-node 3-adapter/xml-json-adapter.ts
  ```

### 4. Library Management System

- **Chạy bài quản lý thư viện (Library System):**
  ```bash
  npx ts-node 4-library-system/index.ts
  ```
